🧹 Καθαρισμός localStorage

Εργαλείο για τον καθαρισμό κακών τιμών από το localStorage

⚠️ Προσοχή: Αυτό το εργαλείο θα διαγράψει κακές τιμές από το localStorage. Κάντε backup πριν συνεχίσετε αν χρειάζεται.
📋 Τι καθαρίζει:
  • Κλειδιά με τιμή '[object Object]'
  • Κακά JSON strings που δεν μπορούν να γίνουν parse
  • Κλειδιά που αρχίζουν με '{' ή '[' αλλά δεν είναι έγκυρα JSON
Object.keys(localStorage).forEach(k => { const v = localStorage.getItem(k); if (v === '[object Object]') localStorage.removeItem(k); else if (typeof v === 'string') { if ((v.startsWith('{') || v.startsWith('['))) { try { JSON.parse(v); } catch { localStorage.removeItem(k); } } } });
🔒 Οδηγίες για Δοκιμή σε Ιδιωτικό Παράθυρο:
  1. Ανοίξτε ένα νέο ιδιωτικό παράθυρο (Ctrl+Shift+N)
  2. Πηγαίνετε στη διεύθυνση: http://localhost:3000/clean-localStorage.html
  3. Εκτελέστε τον καθαρισμό για να σιγουρευτείτε ότι δεν υπάρχουν παρεμβολές από extensions
  4. Τα extensions δεν λειτουργούν σε ιδιωτικό παράθυρο, οπότε θα αποφύγετε τα content.js errors
